Posts must be added to the _posts folder and require a date prefix:
2020-10-01-my-new-post.md

Drafts can be added to the _drafts folder and don't require any date prefix:
my-new-draft.md

To run Jekyll, use the launch config in vscode and access the server in a browser:
http://localhost:4000/

To publish, commit changes on the main branch and deploy it to gh-pages:
git push origin main
npm run publish
